  • Introduction

    • In this task we were calculating velocity of the moving object using the accelerometer on the wiced sense

  • Work done

    • For testing purpose we choose Firebird V as a moving object.We place the wiced sense on it.We programmed the robot to linearly increase the speed and then stop the robot when stop bit is sent via Xbee.The code for Firebird V can found here.The data from the wiced sense was logged in the database and later exported into a CSV file.The data was than processed in matlab and graph of the processed data was plotted.Readings were taken from Firebird V.But as the speed of the robot was slow as compared to real world application,we tested the readings on bike too.

  • Observation

    • Data from sensors accelerometer was mapped when graphs were plotted.We found that there is drift in values after sometime.Even if the velocity is brought to zero the values do not reach to zero

    Another thing we observed that the sensitivity of the accelerometer was low.The procedure to change the sensitivity is explained in programming the wiced sense

  • Future work that can be continued

    • Drift can be reduced by combining the accelerometer and gyroscope readings.

    • Kalman filter can be implemented to improve the result
